    <h1>Store Centre</h1>
    <p>A frontend e-commerce project for Xiaomi products, built with <strong>React.js</strong> and <strong>Tailwind CSS</strong>.</p>

    <h2 id="features">Features</h2>
    <ul>
        <li>Browse Xiaomi products with sleek UI components.</li>
        <li>Product filtering and sorting.</li>
        <li>Responsive design using Tailwind CSS.</li>
        <li>Interactive shopping cart with item management.</li>
    </ul>

    <h2 id="installation">Installation</h2>
    <p>1. Clone the repository:</p>
    <div class="code-block"><code>git clone https://github.com/OscarM03/Store-Centre.git</code><br><code>cd Store-Centre</code></div>
    <p>2. Install dependencies:</p>
    <div class="code-block"><code>npm install</code></div>

    <h2 id="usage">Usage</h2>
    <p>1. Start the development server:</p>
    <div class="code-block"><code>npm start</code></div>
    <p>2. Open <a href="http://localhost:3000">http://localhost:3000</a> to view it in the browser.</p>

    <h2 id="folder-structure">Folder Structure</h2>
    <ul>
        <li><code>src/components</code>: Contains reusable UI components.</li>
        <li><code>src/sections</code>: Sections - Application pages (Hero, Products, Cart).</li>
        <li><code>src/utils</code>: Static data</li>
        <li><code>src/constants</code>: Images export</li>
    </ul>

    <h2 id="technologies-used">Technologies Used</h2>
    <ul>
        <li><strong>React.js</strong>: For UI components and state management.</li>
        <li><strong>Tailwind CSS</strong>: For responsive, utility-first styling.</li>
        <li><strong>JavaScript</strong>: Primary language for functionality.</li>
    </ul>
